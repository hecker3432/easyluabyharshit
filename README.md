# easyluabyharshit
Introduction
This build of lua tools has been created by madoiscool and harshit. This allows you to install any steam game for free with generic fixes or online fixes IF available
Step by Step guide

1.Extract the .rar files and install steam tools(steam should be installed)


2.(Optional) Right click the icon that appears on the screen looking like steam and click Hide floating screen to remove that shitty overlay

3.First run in powershell (admin)

```Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass```
Then

```iex "& { $(irm 'clemdotla.github.io/luatools-installer/install-plugin.ps1') } -DownloadLink 'https://github.com/clemdotla/steamtools-collection/releases/download/Latest/steamtools-collection.zip' -PluginName 'steamtools-collection'"```

4.Now restart steam (startup will take longer) and go to store and search for any game and look for a add through lua tools button and click on it

5. After the proccess is done restart steam and find it in your library

6.After installing the game search for it again in the store and after oppening its details see for a lua tools icon, click on it and click fixes and select generic OR all in one OR online fix (for online)

7. and you are ready to go
